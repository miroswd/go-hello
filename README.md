# Starting with go lang

### Initialize the project

```shell
go mod init project
```

### Run main file

```shell
go run .
```

### Create a executable file

```shell
go build
```

```shell
./go-hello
```



## Operators

`=`  -> Assignment operator, for existents variables

`:=` -> Declaration operator


eg: 

```go
x := 10
x = 20
```
